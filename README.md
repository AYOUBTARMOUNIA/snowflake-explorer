# ❄️ Snowflake Schema Explorer

Application Streamlit pour explorer les tables d'un schéma Snowflake.

## Configuration

1. Copier `.env.example` en `.env` et remplir vos credentials Snowflake
2. Installer les dépendances : `pip install -r requirements.txt`
3. Lancer : `streamlit run app.py`

## Déploiement Streamlit Cloud

1. Connecter ce repo sur [share.streamlit.io](https://share.streamlit.io)
2. Ajouter les secrets Snowflake dans les settings de l'app
